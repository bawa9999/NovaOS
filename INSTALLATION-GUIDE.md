# NovaOS Installation Guide

## Requirements
- Samsung Galaxy A3 2015 (a3xulte)
- TWRP Recovery installed
- Battery charged 50%+
- BACKUP YOUR DATA!

## Download Files
1. NovaOS-1.0-LTS-a3xulte.zip (499MB)
2. NovaOS-Plus-Enhanced.zip (2.1MB)
3. NovaOS-Wallpapers.zip (2.3MB) - Optional
4. NovaOS-Ringtones.zip - Optional

## Installation Steps

### Step 1: Backup
- Boot to TWRP
- Select Backup
- Backup System, Data, Boot
- Save to SD card or PC

### Step 2: Wipe
- Select Wipe → Advanced Wipe
- Check: System, Data, Cache, Dalvik
- Swipe to Wipe

### Step 3: Install NovaOS
1. Install → Select NovaOS-1.0-LTS.zip
2. Swipe to Flash
3. Install → Select NovaOS-Plus-Enhanced.zip
4. Swipe to Flash
5. (Optional) Flash Wallpapers and Ringtones
6. Reboot System

### Step 4: First Boot
- First boot takes 5-10 minutes
- Set up your device
- Enjoy NovaOS!

## Troubleshooting
- **Stuck at boot:** Wipe cache/dalvik and reboot
- **No signal:** Flash stock modem
- **Bootloop:** Restore TWRP backup
