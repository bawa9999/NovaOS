# About NovaOS

## What is NovaOS?
NovaOS breathes new life into older Android devices. We provide long-term support, security updates, and performance optimizations for phones abandoned by manufacturers.

## Our Mission
- Keep older devices secure and usable
- Reduce e-waste
- Provide free, open-source software
- Focus on stability over bleeding-edge features

## Two Editions

### NovaOS Core (LTS)
- 3-year security updates
- Maximum stability
- Minimal bloat
- Perfect for daily drivers

### NovaOS Plus
- All Core features
- Custom UI enhancements
- Extra features
- More customization
