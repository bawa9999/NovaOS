# NovaOS Changelog

## Version 1.0 (January 28, 2026)

### Initial Release
First stable release of NovaOS for Samsung Galaxy A3 2015

### Features
- ✅ Complete rebrand from LineageOS to NovaOS
- ✅ Android 11 (R) base
- ✅ Enhanced security & privacy
- ✅ Performance optimizations
- ✅ Custom boot animation
- ✅ Battery optimizations
- ✅ Custom wallpapers (space theme)
- ✅ Custom ringtones & sounds
- ✅ Firewall & network security
- ✅ Aggressive battery doze mode

### Security
- Monthly security patches
- Enhanced iptables firewall
- Kernel hardening
- ADB secure by default
- Privacy-focused defaults

### Performance
- Optimized dex2oat threads
- Improved boot time
- Better RAM management
- SkiaGL rendering

### Known Issues
- None reported (first release)
